# Mealmaster v0.1.0

First public desktop release of Mealmaster.

## What Changed

- Introduced the Mealmaster desktop build for local household meal planning
- Added a planner-first layout with `Plan`, `Household Settings`, `Recipes`, and `Pantry`
- Added weekly dinner planning with approval and regenerate flows
- Added favorites and planner chat preview/apply support
- Added optional pantry-aware planning
- Added local persistence for household settings, pantry items, recipes, and saved plans

## What To Know Before Updating

- This release is intended for Windows
- Data is stored locally on the machine running the app
- Some planning behavior may differ depending on whether optional AI configuration is enabled
- If you have used earlier internal builds, this release reflects the Mealmaster product naming and planner-first layout

## Known Issues

- Windows may show a trust prompt because the executable is unsigned
- Public desktop packaging is currently Windows-first
- Hosted multi-user sync is not part of this release

## Install

1. Download `Mealmaster.exe`
2. Run it
3. Follow any normal Windows prompts for opening an unsigned app you trust

## Feedback

If you hit a bug or have a feature request, open an issue in the public releases repository.
